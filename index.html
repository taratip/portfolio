HOUSE PRICE ESTIMATOR

Predict house price in Boston Area, Nov 1, 2018

It would be nice to know the price of houses with specic set of features that we already know what kind of houses we want. This project is to estimate a house price based on features such as bedrooms, bathrooms, lot size, etc. A web application is built to get features input from users and it uses Machine learning on publicly available data for house price estimation.

RAINFALL ANALYSIS

A statistical analysis of 30 years Rainfall Data, Nov 7, 2018

The daily rainfall data of 30 years is used to understand normal rainfall. The mean and standard deviation was calculated to check the rainfall variability. The tools used include Python, Flask, Numpy, Panda, and Plotly. The data is available at Thai meteorological centers.
